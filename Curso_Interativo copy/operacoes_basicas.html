Operações Básicas

- Cálculos são feitos de forma direta no console do Python ( Idle ):

>>> 2 + 2
4
>>> 2 + 2 - 1
3
>>> 2 * ( 2 + 3 )
10
>>>( - 7 + 2 ) * ( - 4 )
20

- Tente dividir um número por zero.

>>> 5 / 0
ZeroDivisionError
Leia a indicação de erro com cuidado! Assim poderá reparar rapidamente seu código. Nesse caso, percebemos que existe uma divisão por zero.

- Divisão entre dois inteiros ( utilizando / e // . Perceba a diferença no resultado. Na próxima página verá o porquê)

- Cálculo com dois floats( número com casas decimais )

- Cálculo entre um float e um inteiro produz float.

>>> 10 / 2
5.0
>>> 10 // 2
5
>>> 6.0 * 7.0
42.0
>>> 4 + 3.62
7.62
Um float pode ser adicionado a um inteiro, porque o Python converte o inteiro em float. No entanto, essa conversão implícita é uma exceção. Normalmente, teremos que converter os valores manualmente se quiser operá-los.

Exponenciação

>>> 2 ** 3
8
>>> 9 ** (1 / 2)
3.0

Quociente e Resto

>>> 15 // 6
5
>>>2.5 % 2
0.5
