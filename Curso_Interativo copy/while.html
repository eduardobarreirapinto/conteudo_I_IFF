WHILE

- Bloco de código roda enquanto a expressão for verdadeira.

- No IDLE, abra um novo arquivo ( New File ou Ctrl N ) e digite o código abaixo.

i = 1
while i < = 5 :
    print (' O pai tá ON ! ' )
    i + = 1
print ( ' O pai tá OFF ! ')

- Para executar o código pressione F5.
