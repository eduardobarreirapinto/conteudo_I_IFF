Variáveis

- Permite gravar um valor a um nome por atribuição ( = ).

- A variável pode ser chamada posteriormente no programa.

- A atribuição não provoca nenhuma saída.

>>> x = 39
>>> print ( x )
39
>>> print( x + 10 )
49
>>> print ( x )
39
>>> x = x + 39
>>> print ( x )
49
>>> print( ' Isso agora é um texto -> ' + str ( x ) )
Isso agora é um texto -> 49
>>> num_Serie_Gerador = 12345
>>> num_Serie_Gerador = 54321
>>> print( num_Serie_Gerador )
54321
