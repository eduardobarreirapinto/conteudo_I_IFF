Criando Funções próprias

- Até agora utilizamos as funções nativas do Python. E isso realmente é muito bom.

- Mas também podemos criar nossas próprias funções!!!

- Veja o exemplo abaixo:

def exemplo_Func ( ) :
    print ( "Curso" )
    print ( "Python" )
exemplo_Func ( )
'Curso'
'Python'

- Perceba que é necessário definir uma função antes de chamá-la.

- Caso contrário, tomaremos um erro.

- Veja o exemplo abaixo.

imprime ( )
def imprime ( ) :
    print ( "Curso Python" )
NameError: name 'imprime' is not defined

Criando Funções próprias - Argumentos

- Agora iremos inserir argumentos em uma função.

- Argumentos são os dados inseridos dentro do parênteses da função.

- Podemos inserir um ou mais argumentos de acordo com a necessidade.

- O exemplo abaixo tem apenas um argumento que será concatenado com uma string (!!!).

def exemplo_Func ( arg ) :
    print ( arg + "!!!" )
exemplo_Func ( "Curso Python na Metrologia" )
'Curso Python na Metrologia!!!'

- Vamos criar uma função que some dois números.

def soma ( a , b ) :
    print ( a + b )
soma ( 2 , 4 )
6

- Um erro muito comum que acontece quando criamos uma função é defenirmos uma variável dentro da função e tentar chamar essa variável fora dela.

- Observem esse tipo de erro abaixo para vocês já ficarem espertos quando criarem suas funções.

def exemplo ( a ) :
    x = a
exemplo ( 2 )
print ( 2 )
NameError: name 'x' is not defined
Variável definida dentro da função não pode ser chamada fora !!!

Retorno

- Funções podem retornar informação;

- Geralmente as funções recebem os dados de entrada (argumentos), processam esses dados( fazem cáculo, comparam e etc) e depois retornam o resultado

def maior ( x , y ) :
    if x > y :
        return x
    else :
        return y
z = maior ( 2 , 3 )
print ( z )
3

- Uma vez que o return foi acionado, a função se encerra. Observe o exemplo abaixo:

def soma ( x , y ) :
    return x + y :
    print ( x + y )
A função print não será chamada. O código (dentro de uma função) após o return ter sido acionado não será mais lido pelo interpretador Python. Mas não fique triste por isso. Você vai superar! Vida que segue!

Função dentro de função

- Função dentro de função ???? Que loucura!!!

- Mas não é loucura não. Muito pelo contrário. Muito útil! Reparem:

def soma ( x , y ) :
    return x + y :
def soma2vezes ( soma, x , y ) :
    return ( soma (x + y) ,soma (x + y) ):
print (soma2vezes( soma , 1 + 2 ))
6

- Passar a função como parâmetro é uma boa prática. Mas não é necessário. O código abaixo também funcionan:

def soma ( x , y ) :
    return x + y :
def soma2vezes ( x , y ) :
    return ( soma (x + y) ,soma (x + y) ):
print (soma2vezes( 1 + 2 ))
6
Posso criar uma função e utilizá-la dentro de uma outra função. Passando uma função como parâmetro de outra - ex: soma2vezes(soma,1,2) - podemos saber de cara saber quais são os requisitos necessários para essa outra função funcionar. Daí ser uma boa prática de organizar e dar manutenção ao seu código.
