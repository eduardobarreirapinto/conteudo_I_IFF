Definição

Python é uma linguagem de programação de alto nível, com aplicações em diversas áreas:

  1. Programação web,
  2. Aplicativos Mobile,
  3. Scripts,
  4. Ciência de Dados,
  5. Aprendizado de máquina,
  6. Inteligência artificia,
  7. Computação científica

É muito popular e usado por organizações como Google, NASA, CIA e Disney.

E a partir de agora o Inmetro!!!

Python é processado em tempo de execução pelo interpretador.

Um interpretador é um programa que executa scripts escritos em uma linguagem interpretada como o Python.

Não há necessidade de compilar seu programa antes de executá-lo !

As duas principais versões do Python são 2.x e 3.x.

Estes são subdivididos em versões menores, como 2.7, 3.7 e 3.8.

Código escrito para o Python 3.x é garantido para funcionar em todas as versões futuras. Python 2 está sendo descontinuado.

Tanto a versão 2.x quanto a 3.x do Python são usadas atualmente.

Não é difícil mudar de uma versão para outra. Não é necessário saber a versão 2 para começcar a aprender a versão 3.

Na documentação, o código Python geralmente contém referências ao grupo de comédia Monty Python.
