Criando Funções próprias

- Até agora utilizamos as funções nativas do Python. E isso realmente é muito bom.

- Mas também podemos criar nossas próprias funções!!!

- Veja o exemplo abaixo:

def exemplo_Func ( ) :
    print ( "Curso" )
    print ( "Python" )
exemplo_Func ( )
'Curso'
'Python'

- Perceba que é necessário definir uma função antes de chamá-la.

- Caso contrário, tomaremos um erro.

- Veja o exemplo abaixo.

imprime ( )
def imprime ( ) :
    print ( "Curso Python" )
NameError: name 'imprime' is not defined

Criando Funções próprias - Argumentos

- Agora iremos inserir argumentos em uma função.

- Argumenros são os dados inseridos dentro do parênteses da função.

- Podemos inserir um ou mais argumentos de acordo com a necessidade.

- O exemplo abaixo tem apenas um argumento que será concatenado com uma string (!!!).

def exemplo_Func ( arg ) :
    print ( arg + "!!!" )
exemplo_Func ( "Curso Python na Metrologia" )
'Curso Python na Metrologia!!!'

- Vamos criar uma função que some dois números.

def soma ( a , b ) :
    print ( a + b )
soma ( 2 , 4 )
6

- Um erro muito comum que acontece quando criamos uma função é defenirmos uma variável dentro da função e tentar chamar essa variável fora dela.

- Observem esse tipo de erro abaixo para vocês já ficarem espertos quando criarem suas funções.

def exemplo ( a ) :
    x = a
exemplo ( 2 )
print ( 2 )
NameError: name 'x' is not defined
Variável definida dentro da função não pode ser chamada fora !!!

Concatenação de Strings

- Assim como números, os textos (Strings) também podem adicionar outros textos - Concatenação;

- Não importa se os textos utilizam aspas simples ou duplas;

>>> “Eduardo” + ‘Pinto’
'Eduardo Pinto'
>>> print ( '2' + " + " + '2')
2 + 2
>>> "1" + 1
TypeError: can only concatenate str (not "int") to str
Não é possível somar uma String com um número ( Inteiro ou Float ). Será que podemos multiplicar uma String com um número inteiro??? Vamos em frente para ver.

Operação com Strings

- Multiplicação : Repete-se a String pela quantidade multiplicada

- A ordem não importa.

>>> print ( '2' * 4 )
2222
>>> "22" * "44"
TypeError: can't multiply sequence by non-int of type 'str'
>>> "Metrologia" * 4.0
TypeError: can't multiply sequence by non-int of type 'float'

Qual a saída do código :

>>> print ( 'Vasco campeão em ' + 2023 )
