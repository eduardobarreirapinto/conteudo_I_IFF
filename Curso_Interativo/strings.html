Strings

- Texto (sequência de caracteres) em Python.

- Uma sequência é criada inserindo o texto entre duas aspas simples ou duplas.

>>> 'Curso de Python com aspas simples'
'Curso de Python com aspas simples'
>>> "Curso de Python com aspas duplas"
'Curso de Python com aspas simples'

- Alguns caracteres não podem ser incluídos diretamente em uma String.

- Por exemplo, as aspas duplas não podem ser incluídas diretamente em uma cadeia de aspas duplas. Isso faria com que terminasse prematuramente.

- Caracteres como esses devem ser evitados colocando uma barra invertida diante deles.

- Outros caracteres comuns que devem ser escapados são novas linhas e barras invertidas.

- Aspas duplas só precisam ser escapadas em strings de aspas duplas, e o mesmo é verdadeiro para strings de aspas simples.

>>> 'He\'s not the doctor. He\'s an excellent software developer !'
'He's not the doctor. He's an excellent software developer!'

Nova Linha

- \n representa pular linha(nova linha).

- Para evitar escrever manualmente "\ n" para pular linhas em uma string, utiliza-se uma string com três conjuntos de aspas e as novas linhas que são criadas pressionando Enter.

>>> ' ' ' Estou escrevendo e apertei o enter
e voltei a escrever aqui. ' ' '
‘Estou escrevendo e apertei o enter\n e voltei a escrever aqui.'

Concatenação de Strings

- Assim como números, os textos (Strings) também podem adicionar outros textos - Concatenação;

- Não importa se os textos utilizam aspas simples ou duplas;

>>> “Eduardo” + ‘Pinto’
'Eduardo Pinto'
>>> print ( '2' + " + " + '2')
2 + 2
>>> "1" + 1
TypeError: can only concatenate str (not "int") to str
Não é possível somar uma String com um número ( Inteiro ou Float ). Será que podemos multiplicar uma String com um número inteiro??? Vamos em frente para ver.

Operação com Strings

- Multiplicação : Repete-se a String pela quantidade multiplicada

- A ordem não importa.

>>> print ( '2' * 4 )
2222
>>> "22" * "44"
TypeError: can't multiply sequence by non-int of type 'str'
>>> "Metrologia" * 4.0
TypeError: can't multiply sequence by non-int of type 'float'

Qual a saída do código :

>>> print ( 'Vasco campeão em ' + 2021 )
