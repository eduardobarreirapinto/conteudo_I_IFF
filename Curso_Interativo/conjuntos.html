O que são Conjuntos em Python?

- Conjuntos são estruturas de dados em Python que armazenam coleções não ordenadas de itens únicos.

- Eles são mutáveis, ou seja, seus valores podem ser alterados após a criação.

- Veja um exemplo simples de um conjunto em Python:

conjunto = {1, 2, 3, 'a', 'b', 'c'}

- Conjuntos são definidos usando chaves `{}` e itens separados por vírgula.

Operações com Conjuntos

- Conjuntos suportam operações matemáticas como união, interseção e diferença.

- Exemplo de união de conjuntos:

conjunto1 = {1, 2, 3}
conjunto2 = {3, 4, 5}
print(conjunto1 | conjunto2) // Saída: {1, 2, 3, 4, 5}

- Exemplo de interseção de conjuntos:

conjunto1 = {1, 2, 3}
conjunto2 = {3, 4, 5}
print(conjunto1 & conjunto2) // Saída: {3}

Uso de Conjuntos

- Conjuntos são frequentemente usados para remover duplicatas de uma sequência.

- Exemplo de uso de conjunto para remover duplicatas:

lista = [1, 2, 2, 3, 4, 4, 5]
conjunto = set(lista)
print(conjunto) // Saída: {1, 2, 3, 4, 5}

Aplicações dos Conjuntos

- Conjuntos são úteis para operações de pertença e eliminação de duplicatas.

- Exemplo de uso de conjuntos em Python:

alunos = {'João', 'Maria', 'José'}
alunos_presentes = {'Maria', 'José'}
print(alunos - alunos_presentes)
// Saída: {'João'}

Iterando em Conjuntos

- É possível iterar sobre os elementos de um conjunto usando um loop `for`.

- Exemplo de iteração em um conjunto:

conjunto = {1, 2, 3, 'a', 'b', 'c'}
for item in conjunto:
    print(item)

- Isso imprimirá cada elemento do conjunto em uma nova linha.
