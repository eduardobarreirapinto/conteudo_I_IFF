O que são Tuplas em Python?

- Tuplas são estruturas de dados em Python semelhantes a listas.

- Elas são imutáveis, ou seja, uma vez criadas, não podem ser modificadas.

- Veja um exemplo simples de uma tupla em Python:

tupla = (1, 2, 3, 'a', 'b', 'c')

- Tuplas são definidas usando parênteses `()` e itens separados por vírgula.

Operações com Tuplas

- Assim como listas, tuplas suportam indexação e fatiamento.

- Exemplo de indexação em uma tupla:

tupla = (1, 2, 3, 'a', 'b', 'c')
print(tupla[0]) // Saída: 1

- Exemplo de fatiamento em uma tupla:

tupla = (1, 2, 3, 'a', 'b', 'c')
print(tupla[1:4]) // Saída: (2, 3, 'a')

Uso de Tuplas

- Tuplas são frequentemente usadas para retornar múltiplos valores de funções.

- Exemplo de retorno de tupla em uma função:

def retorna_coordenadas():
    return (10, 20)
x, y = retorna_coordenadas()
print(x, y) // Saída: 10 20

Aplicações das Tuplas

- Tuplas são úteis em situações onde os dados não devem ser modificados após a criação.

- Exemplo de uso de tuplas em Python:

dimensoes = (800, 600)
cores = ('vermelho', 'verde', 'azul')
print(f"Tamanho da imagem: {dimensoes[0]}x{dimensoes[1]} pixels")
print(f"Cores disponíveis: {', '.join(cores)}")
// Tamanho da imagem: 800x600 pixels
// Cores disponíveis: vermelho, verde, azul

Iterando em Tuplas

- É possível iterar sobre os elementos de uma tupla usando um loop `for`.

- Exemplo de iteração em uma tupla:

tupla = (1, 2, 3, 'a', 'b', 'c')
for item in tupla:
    print(item)

- Isso imprimirá cada elemento da tupla em uma nova linha.
