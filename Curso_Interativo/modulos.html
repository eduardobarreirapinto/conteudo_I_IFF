Módulos em Python

- Um módulo é uma maneira de organizar a sua função e outros programas para utilizar reutiliza.

- E isso realmente é muito bom.

- Mas também podemos criar nossos próprios módulos!!!

- Veja o exemplo abaixo:

def exemplo_Func ( ) :
    print ( "Curso" )
    print ( "Python" )
exemplo_Func ( )
'Curso'
'Python'

- Perceba que é necessário definir uma função antes de chamá-la.

- Caso contrário, tomaremos um erro.

- Veja o exemplo abaixo.

imprime ( )
def imprime ( ) :
    print ( "Curso Python" )
NameError: name 'imprime' is not defined

Módulos em Python - Importando e Reutilizando Código

- Agora iremos inserir importando e reutilizando código.

- Uma das maneiras mais poderosas de usar módulos em Python é através do comando import.

- Podemos importar módulos padrão do Python ou módulos que criamos.

- O exemplo abaixo demonstra como importar um módulo próprio chamado meu_modulo.py:

import meu_modulo
meu_modulo.minha_funcao()

- Isso executará a função minha_funcao() definida no arquivo meu_modulo.py.

- Com módulos, podemos organizar melhor nosso código, reutilizar funcionalidades em diferentes partes do projeto e facilitar a manutenção.

Exercício: Criando e Utilizando Módulos

- Agora é sua vez de praticar! Crie um arquivo chamado calculos.py com as seguintes funções:

  • funcao_soma(a, b) - Retorna a soma de a e b.
  • funcao_subtracao(a, b) - Retorna a subtração de a por b.
  • funcao_multiplicacao(a, b) - Retorna o resultado da multiplicação de a por b.
  • funcao_divisao(a, b) - Retorna o resultado da divisão de a por b.

- Depois, crie um arquivo chamado main.py que importa o módulo calculos.py e utiliza todas as funções definidas.

- Execute o main.py para verificar se as funções estão funcionando corretamente.

Documentando Módulos em Python

- Documentar módulos em Python é uma boa prática para facilitar o entendimento e o uso do código por outros desenvolvedores (ou por você mesmo no futuro).

- Utilize docstrings para descrever o que cada função faz, quais são os parâmetros esperados e qual é o valor de retorno.

- Veja um exemplo de como documentar uma função:

def soma(a, b):
    """Retorna a soma de dois números."""
    return a + b

- Ao documentar suas funções, você ajuda a si mesmo e aos outros a entenderem melhor o propósito e o uso de cada função no seu código.

Conclusão

- Módulos em Python são poderosos para organizar, reutilizar e documentar seu código.

- Com eles, você pode criar bibliotecas de funções que podem ser compartilhadas e utilizadas em diferentes projetos.

- Lembre-se de importar os módulos que você precisa e documentar suas funções para facilitar a manutenção e o trabalho em equipe.
