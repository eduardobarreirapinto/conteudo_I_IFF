FOR

- Percorre um objeto iterável.

- Exemplo de objetos iteráveis: Listas, Strings, e etc.

- Vamos percorrer uma lista e imprimir seu elementos.

lista = [ 'Eu' , 'sou' , 'Desenvolvedor' , 'Python' ]
for i in lista :
    print ( i )

- O loop For itera sob a lista imprimindo todos os itens:

Eu
sou
Desenvolvedor
Python

- Legal, mas como faço para a frase ficar na mesma linha ? ? ?.

A função print ( ) possui um parâmetro default end = '\n'.

- Ok, vamos mudar esse parâmetro por um simples espaço em branco. Veja como fica:

lista = [ 'Eu' , 'sou' , 'Desenvolvedor' , 'Python' ]
for i in lista :
    print ( i , end = ' ')

- O loop For itera sob a lista imprimindo todos os itens:

Eu sou Desenvolvedor Python

- Parabéns! Você é um desenvolvedor Python ! Mas ainda temos um bom caminho a seguir. Vamos lá.

FOR

- Acabamos de iterar sob uma lista. Mas lembre que a função .range( ) devolve um objeto iterável.

- Vamos iterar dob ela então:

for i in range ( 3 ) :
    print ( ' oi ' )

- O loop For itera sob a lista imprimindo todos os itens:

oi
oi
oi

- Mais um exemplo:

for i in range ( 3 ) :
    print ( i )

- Itera sob o própio objeto range:

0
1
2
A função .range( ) não necessita ser convertida para lista quando utilizada dentro do Loop For.

Listas e Strings

- String pode ser interpretada como uma lista de caracteres IMUTÁVEIS

- Vamos iniciar utilizando uma lista

- Podemos selecionar parte dessa lista através do fatiamento.

- No fatiamento o primeiro parâmetro é incluso e o segundo não incluso.

lista = [ 'Eu' , 'sou' , 'bonitão' ]
print ( lista[ 1 ] )
print ( lista[ 0 : 2 ] )
print ( lista[ : 2 ] )
print ( lista[ 1 : 2 ] )
print ( lista[ 1 : ] )
print ( lista[ : ] )

- Observe o resultado e reflita.

sou
['Eu', 'sou']
['Eu', 'sou']
['sou']
['sou', 'bonitão']
['Eu', 'sou', 'bonitão']

- Bonitão, vamos mudar para string agora.

frase = 'Eu sou bonitão'
print ( frase[ 1 ] )
print ( frase[ 0 : 6 ] )
print ( frase[ : 6 ] )
print ( frase[ 7 : 13 ] )
print ( frase[ 7 : ] )

- Resultado será semelhante ao comportamento de uma lista.

u
Eu sou
Eu sou
bonitã
bonitão
