Booleans

- Outro tipo : True e False

- Pode ser criado comparando valores ( == )

>>> verdadeiro = True
>>> verdadeiro
True
>>> 47 == 47.0
True
>>> 47 == '47'
False
Não confundir == ( comparação ) com = ( atribuição)

Booleans

Não igual !=

- Terá uma saída True se valores ou tipos forem diferentes.

>>> 5 != 5
False
>>> 47 != 47.0
False
>>> 47 != '47.0'
True

Comparação

>>> 25 >= 25
True
>>> 25 > 25
False

Lógica Booleana and / or / not:

and Para a saída ser positiva, todos os termos precisam ser positivos

>>> 10 == 10 and 22 == 22
True
>>> 10 == 10 and 22 == 23
False
>>> 1 != 1 and 2 == 2
False
>>> 4 < 2 and 5 > 9
False

or Para a saída ser positiva, pelo menos um dos termos precisa ser positivo

>>> 10 == 10 or 22 == 22
True
>>> 10 == 10 or 22 == 23
True
>>> 1 != 1 or 2 == 2
True
>>> 4 < 2 or 5 > 9
False

not Inverte o valor booleano do termo

>>> not 22 == 22
False
>>>not ( ( 22 == 22 ) or ( 22 == 23 ) )
False

Algebra Booleana

>>> False == False or True
True
>>> True == (False or True )
True
>>> ( True == True ) or False
True

Qual a saída da expressão :

>>> ( False == False or True ) or ( 44 == 44 and 'b' > 'a')
