O que são Dicionários em Python?

- Dicionários são estruturas de dados em Python que armazenam pares de chave-valor.

- Eles são mutáveis, ou seja, seus valores podem ser alterados após a criação.

- Veja um exemplo simples de um dicionário em Python:

dicionario = {'nome': 'João', 'idade': 25, 'cidade': 'São Paulo'}

- Dicionários são definidos usando chaves `{}` e pares de chave-valor separados por dois pontos `:`.

Operações com Dicionários

- Você pode acessar os valores em um dicionário usando suas chaves.

- Exemplo de acesso a valores em um dicionário:

dicionario = {'nome': 'João', 'idade': 25, 'cidade': 'São Paulo'}
print(dicionario['nome']) // Saída: João

- Exemplo de modificação de valores em um dicionário:

dicionario['idade'] = 26
print(dicionario['idade']) // Saída: 26

Uso de Dicionários

- Dicionários são frequentemente usados para armazenar dados relacionados.

- Exemplo de uso de dicionário em uma função:

def cria_dicionario(nome, idade, cidade):
    return {'nome': nome, 'idade': idade, 'cidade': cidade}
dados = cria_dicionario('João', 25, 'São Paulo')
print(dados) // Saída: {'nome': 'João', 'idade': 25, 'cidade': 'São Paulo'}

Aplicações dos Dicionários

- Dicionários são úteis para organizar dados de forma clara e acessível.

- Exemplo de uso de dicionários em Python:

aluno = {'nome': 'Maria', 'idade': 22, 'curso': 'Engenharia'}
professor = {'nome': 'Carlos', 'idade': 40, 'disciplina': 'Matemática'}
print(aluno)
print(professor)
// Saída: {'nome': 'Maria', 'idade': 22, 'curso': 'Engenharia'}
// Saída: {'nome': 'Carlos', 'idade': 40, 'disciplina': 'Matemática'}

Iterando em Dicionários

- É possível iterar sobre os pares de chave-valor de um dicionário usando um loop `for`.

- Exemplo de iteração em um dicionário:

dicionario = {'nome': 'João', 'idade': 25, 'cidade': 'São Paulo'}
for chave, valor in dicionario.items():
    print(f'{chave}: {valor}')

- Isso imprimirá cada par de chave-valor em uma nova linha.
