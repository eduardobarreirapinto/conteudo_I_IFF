Saída

- Os programas recebem dados , processam e produzem uma saída.

- A função de impressão print ( ) produz uma saída.

- Exibe uma representação textual de algo na tela.

>>> print ( 7 + 6 )
13
>>> print(' Olá\nInmetro! ')
Olá
Inmetro!
Reparem que com a função print as aspas somem!

Entrada

- A função input ( ) pega uma entrada do usuário.

- E a função retorna o que ele escreveu como uma String (automaticamente 'escapada');

>>> print('Qual é o seu nome?')
Qual é o seu nome? Meu nome é\nEduardo
'Meu nome é\\nEduardo'
O texto dentro do parênteses de uma função é chamado de argumento. Mais informações no capítulo sobre funções.

Continuação

- PERGUNTA: Como fazer ' 4 ' + ' 2 ' ser igual a ' 6 '?

>>>'4' + '2'
'42'

- RESPOSTA: Utilizando as funções de conversão.

>>> str ( int ( '4' ) + int ( '2' ) )
'6'

Converter para String ( texto ) str ( )

Converter para Float ( decimal ) float ( )

Converter para Integer ( inteiros ) int ( )

Qual o resultado da expressão:

>>> float ( input ( 'Valor 1: ' )) + float ( input ( 'Valor 2: ' ))
