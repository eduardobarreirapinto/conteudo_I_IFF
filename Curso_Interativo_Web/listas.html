Listas

- É um tipo de objeto em Python. Veremos o que é objeto em capítulos futuros.

- Uma lista serve para gravar itens de forma indexada .

- Para recuperar um item, é necessário indicar o índice entre colchetes.

- Para substituir um item, é necessário atribuir outro item na posição apontada pelo índice ( entre colchetes ).

- Observe a lista com três strings.

lista = [' Laeta ' , ' Diavi ' , ' Dimci ']
print ( lista [ 0 ] )
print ( lista [ 1 ] )
print ( lista [ 2 ] )
lista [ 0 ] = ' Lavib '
print ( lista )

Métodos

- Você não sabe o que é um objeto ainda. Ok. Sem problemas. Mas já sabe que uma lista serve para armazenar itens.

- Um objeto tem a capacidade de executar funções. Essa funções são chamadas de métodos.

- Assim, uma lista, por ser um objeto, também é capaz de executar funções através de seus métodos. Observe.

y = [ 1 , 2 , 3 , 4 , 5 , 6 ]
y.append( 7 )
print ( y )

- O médodo .append( ) insere um item no final da lista.

[ 1 , 2 , 3 , 4 , 5 , 6 , 7 ]

- O médodo .insert( ) insere um item em qualquer posição da lista. É necessário indicar o índice desejado.

y = [ 1 , 2 , 3 , 4 , 5 , 6 , 7]
y.insert( 6 , 6.5 )
print ( y )
[ 1 , 2 , 3 , 4 , 5 , 6 , 6.5 , 7 ]

- E o médodo index utilizado abaixo. Tente advinha o que ele faz.

y = [ 1 , 2 , 3 , 4 , 5 , 6 , 7]
print ( y.index( 3 ) )

- Fácil né. Devolve o índice da primeira ocorrência que acha um item de valor 3.

2

Range

- É um função ' built-in ' que devolve uma sequência de números.

- Geralmente ela é utilizada em conjunto com a função ' built-in ' de conversão para lista list ( ).

lista = list ( range ( 5 ) )
print ( lista )
[ 0 , 1 , 2 , 3 , 4 ]

- Sem list ( ) a função range ( ) devolve apenas o objeto iterável. Ficará mais claro na próxima seção de LOOP FOR.

print ( range ( 5 ) )
range(0, 5)

- Com dois argumentos range ( início incluso , fim não incluso )

- Observe

print ( list (range ( 2 , 6 ) ) )
[2, 3, 4, 5]

- Podemos configurar o intervalo do range e a sequência dos passos.

- Três argumentos range ( início incluso , fim não incluso , passos )

print ( list (range ( 0 , 10 , 2 ) ) )
[0, 2, 4, 6, 8]
IMPORTANTE: o primeiro elemento da lista tem índice ZERO.
