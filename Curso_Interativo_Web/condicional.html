IF

- Se uma expressão for avaliada como True, algumas instruções serão executadas. Caso contrário, eles não são realizados.

- No IDLE, abra um novo arquivo ( New File ou Ctrl N ) e digite o código abaixo.

if 2 > 1 :
    print (' 2 é maior que 1 ' )
    print ( ' Fim do programa! ')

- Para executar o código pressione F5.

2 é maior que 1
Fim do programa!

- Outro exemplo com IF aninhado. Copie e cole o código abaixo.

valor = 103
if valor > 100 :
    print ( ' Maior que 100 ' )
    if valor <= 105 :
        print ( ' Entre 100 e 105 ' )

- Pressione F5.

Maior que 100
Entre 100 e 105
Python não utiliza { } para separar blocos de códigos como em outras linguagens de progração. No Python, é obrigatório o uso de INDENTAÇÃO ( convencionalmente usa-se 4 espaços ou TAB ).
Não se esqueça do : após a expressão.

IF e ELSE possuem a mesma indentação.

ELIF

- Observe o código abaixo:

x = 341
if x == 300 :
    print ( ' X vale 300 ' )
else :
    if x == 350 :
        print ( ' X vale 350 ' )
    else :
        if x == 341 :
        ----print ( ' X vale 341 ' )
X vale 341

- A sequência Else IF é comum e permitida. Mas o código, devido a indentação, pode ficar confuso e 'inclinado' demais. Não acham?

- No Python, existe a expressão ELIF que substitui o ELSE IF. Assim, o código fica mais limpo e facilita a manutenção.
